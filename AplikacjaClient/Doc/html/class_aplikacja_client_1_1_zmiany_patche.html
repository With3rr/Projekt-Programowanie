Detailed Description

Klasa ZmianyPatche określa kolejno wprowadzone Zmiany w aplikacji przez zespól za to odpowiedzialny

Na podstawie tej klasy twożone są obiekty które wykorzystywane są do dokumentacji zmian wprowadzanych w aplikacji a ich wyniki są wyświetlane w samej aplikacji. Dostęp do nich jest swobodny.
