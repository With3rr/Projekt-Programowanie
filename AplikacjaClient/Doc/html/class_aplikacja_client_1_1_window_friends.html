Detailed Description

Klasa WindowsFriend jako klasa określająca Okno aplikacji do zarządzania urzytkownikami(przyjaciumi).

Klasa ta określa aplikację która pozwala na dostęp do bazy danych i określanie przyjaciół danego zalogowanego do aplikacji urzytkownika. Kolejnymi możlwościami będzie możliwość dodawania znajomych oraz ich usuwania.

metoda która inicjalizuje listę "listaprzyjaciol" objektami typu Friend
