Detailed Description

Klasa GryBiblioteka określa obiekty które wchodzą w skład biblioteki gier (które to dostępne są po zakupieniu przez urzytkownika)

Klasa ta pozwala dostęp do poszczególnych właściwości które są cechami określonego obiektu a w tym przypadku zakupionej gry i będącej w bibliotece urzytkownika Dostęp do nich jest swobodny.
