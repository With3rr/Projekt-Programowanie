Detailed Description

Klasa GryBiblioteka określa obiekty które będą możlwe do przeglądania przez urzytkownika i możlwe do kupienia.

Klasa ta pozwala dostęp do poszczególnych właściwości obiektów typu Gry które będą zawierać podstawowe własciowsci i typy gier sprzedawanych. Dostęp do nich jest swobodny.
